# Frontend - Visualización de Demanda

Este frontend fue desarrollado con **React** para mostrar gráficas de demanda

## ⚙️ Tecnologías
- React
- Bootstrap (para estilos)
- Chart.js (`react-chartjs-2`)
- Axios

## 📌 Funcionalidades
- Selector de rango de fechas (15, 30, 60 días)
- Gráfica interactiva con paginación
- Carga de datos desde una API externa
- Spinner de carga mientras llegan los datos

## ▶️ Cómo ejecutar
1. Instalar dependencias:
   npm install

2. correr proyecto:
   npm start
